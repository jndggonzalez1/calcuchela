# Calcuchela

Calculadora web de una sola página construida con Next.js 14, Tailwind CSS y TypeScript. Diseñada para deployar en Vercel.

## Stack

- **Next.js 14+** con App Router
- **Tailwind CSS**
- **TypeScript**
- **Fuente Inter** (Google Fonts via `next/font`)

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Deploy

Conecta el repositorio en [Vercel](https://vercel.com) y hace deploy automáticamente en cada push a `main`.
